## 参考答案与解析

### 一、填空题

1. **1946、十**（ENIAC 还不属于冯氏机，它是十进制）
2. **存储程序**（核心中的核心）
3. **运算器、控制器**
4. **中央处理器（CPU）**
5. **CPU、主存储器**（老师反复强调，主机=CPU+主存，不含辅存/硬盘）
6. **硬件系统、软件系统、等价性**
7. **Cache（高速缓冲存储器）**
8. **微程序级（或硬联逻辑级）、虚拟**
9. **MIPS、MFLOPS**
10. **2**（Cycles Per Instruction）

### 二、选择题

1. **D** - 存储程序控制是冯·诺依曼机的核心特征
2. **A** - 主机=CPU+主存，内存条是主存的物理体现，硬盘属于辅存
3. **C** - 进位链是硬件实现的细节，对程序员透明
4. **B** - CPI = 总时钟周期数 / 指令数 = (0.01s / 1ns) / 10^6 = 10
5. **B** - 软件功能硬化（如浮点运算器）
6. **C** - 系统总线连接五大部件
7. **B** - Intel x86 是 CISC 的代表
8. **B** - 寄存器在 CPU 内部，速度最快
9. **B** - DBMS 属于系统软件
10. **D** - 1GB = 2^30 字节

## 重点提示

1. **主机定义**：主机 = CPU + 主存，**不含**辅存（硬盘）
2. **软件硬化**：将软件功能用硬件实现（如浮点运算器）
3. **透明性**：对程序员不可见的硬件细节（如进位链）
4. **存储层次**：寄存器 > Cache > 主存 > 辅存
5. **单位换算**：计算机中按 1024（2^10）计算，1GB = 2^30 字节
