## 本章概述

主存储器是计算机系统中用于存放程序和数据的重要部件，是CPU直接访问的存储空间。本章系统地介绍了：

1. **主存的组织结构** - 存储体的构成、基本术语和技术指标
2. **数据的存储方式** - 字节序（大小端）和边界对齐
3. **半导体存储器** - SRAM、DRAM的工作原理及刷新机制，ROM的分类
4. **主存的扩展** - 位扩展、字扩展和字位同时扩展，地址译码与片选
5. **性能优化技术** - SDRAM、DDR系列技术的发展

## 学习重点

- 存储容量的计算和存取速度指标（存取时间、存取周期、带宽）
- 大小端字节序的判断和内存布局
- 边界对齐的原理和空间利用率计算
- SRAM与DRAM的区别及DRAM的三种刷新方式
- 主存扩展的连接方法（位扩展、字扩展）和地址分配
- 地址译码器的设计和片选方法（线选法、全译码法、部分译码法）
- SDRAM和DDR系列技术的核心特点

## 难点提示

- **DRAM刷新机制**：理解集中刷新、分散刷新和异步刷新三种方式的工作原理和优缺点
- **主存扩展**：能够根据芯片规格设计扩展方案，正确连接地址线、数据线和控制线，并计算地址范围
- **地址译码**：掌握全译码法和部分译码法的区别，理解地址重叠问题
