# 8.2 外部设备 (External Devices)

> **学习状态**：自学内容（PPT重点）
> **核心考点**：磁盘容量与存取时间计算、显存容量计算。

---

## 1. 硬盘存储器 (**计算题高频点**)

### 1.1 信息分布

*   **记录面**：磁盘的上下两个面
*   **圆柱面 (Cylinder)**：所有记录面上相同半径的磁道组成的圆柱
*   **磁道 (Track)**：每个记录面上的同心圆
*   **扇区 (Sector)**：磁道被等分为若干扇区

### 1.2 地址格式

*   驱动器号、圆柱面号、磁头号、扇区号

### 1.3 存取时间 ($T_a$) —— **必考公式**

**公式**：

$$T_a = T_s + T_w + T_t$$

*   **$T_s$ (寻道时间)**：磁头移动到目标磁道的时间（取平均值）
*   **$T_w$ (旋转延迟)**：扇区转到磁头下的时间（**通常取旋转一周时间的一半**）
*   **$T_t$ (传输时间)**：读写数据的时间（通常忽略不计）

**重点**：
*   题目中若给出转速（如7200转/分），需先算出旋转一周的时间，再除以2得到平均等待时间
*   旋转一周时间 = $\frac{60}{转速}$（秒）
*   平均旋转延迟 = $\frac{1}{2} \times \frac{60}{转速}$（秒）

### 1.4 RAID (磁盘阵列)

*   **核心思想**：
    *   并行传输（提高速度）
    *   冗余校验（提供容错）
*   **优势**：成本低、功耗小、速率高、可靠性高

## 2. 显示设备

### 2.1 VRAM (显存) 容量计算 —— **必考公式**

**公式**

$$\text{VRAM容量} = \text{分辨率} \times \text{颜色深度}$$

*   **分辨率**：如 $1024 \times 768$
*   **颜色深度**：与颜色数有关，$\text{颜色深度} = \log_2(\text{颜色数})$（单位：位）

**常见颜色深度**：
*   256种颜色 = 8位深度 (1字节)
*   真彩色 (24位) = 3字节
*   32位色 = 4字节（包含Alpha通道）

**注意**：
*   如果题目问的是字节数，需要将位数除以8
*   如果涉及双缓冲，需要乘以2

### 2.2 显示刷新率

*   刷新率：每秒刷新屏幕的次数（如60Hz、120Hz）
*   显存带宽需求 = 分辨率 × 颜色深度 × 刷新率

## 💡 学习要点

### 磁盘存取时间计算步骤

1.  确定寻道时间 $T_s$（通常题目给出平均值）
2.  计算旋转延迟 $T_w$：
    *   旋转一周时间 = $\frac{60}{转速}$（秒）
    *   平均旋转延迟 = $\frac{1}{2} \times \frac{60}{转速}$（秒）
3.  计算传输时间 $T_t$（通常忽略或题目给出）
4.  求和：$T_a = T_s + T_w + T_t$

### 显存容量计算注意事项

*   注意单位换算（位 → 字节）
*   如果涉及双缓冲，需要乘以2
*   颜色深度 = $\log_2(\text{颜色数})$，需要记住常见值

### 考试重点

*   **必背**：磁盘存取时间公式、显存容量公式
*   **理解**：磁盘信息分布结构、RAID的基本思想
*   **应用**：能够根据题目条件计算磁盘存取时间或显存容量
